W3C & HTML & CSS

The W3C is the World Wide Web Consortium and their mission is to develop qualified web policies. This company was started by Tim Berner Lee and he was also the director. This organization is made up of a huge team and they make sure they are effectively developing reliable web standards. Further , HTML is the HyperText Markup language. This basically the language to tell the web browser to make the page layout. This includes changing text and adding photos. Images have separate urls and if they are a lot then there are multiple server requests. The CSS is known as cascading style sheets. This is intertwined with HTML documents. It can take the elements inside a HTML and enhance the layout of the web page. CSS is efficient in conserving time to allow more work to get down. Different style sheets are generated and this is useful because this allows people to be more creative when attempting to make a website.
